üìå JavaScript: By Value vs By Reference

JavaScript menangani data berdasarkan tipe: primitif disalin by value, sedangkan non-primitif seperti objek disalin by reference.

‚úÖ By Value (Salin Nilai)

  • Berlaku untuk tipe data primitif: number, string, boolean, null, undefined, symbol, bigint.
  • Nilai disalin, tidak terhubung satu sama lain.
  • Perubahan pada salinan tidak memengaruhi aslinya.
let a = 5;
let b = a;
b = 10;
console.log(a); // 5
a = 5
b = 10
a tetap = 5

‚úÖ By Reference (Salin Referensi)

  • Berlaku untuk tipe non-primitif: object, array, function.
  • Yang disalin adalah referensinya (alamat di memori).
  • Perubahan pada salinan akan memengaruhi data asli.
let obj1 = { name: "Nabiel" };
let obj2 = obj1;
obj2.name = "Ulil";
console.log(obj1.name); // "Ulil"
obj1.name = "Ulil"
karena obj1 dan obj2 menunjuk ke objek yang sama

- Array dan function juga termasuk object, jadi berlaku by reference.

- Variabel tidak menjadi alias langsung, tapi menunjuk ke data yang sama di memory.
